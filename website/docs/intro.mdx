# Formora

**Formora** is a tiny but powerful form engine for React, designed for **predictable validation**, **safe async handling**, and **complex realâ€‘world forms**.

Formora focuses on _clarity and correctness_ instead of magic. Every behavior is explicit, testable, and easy to reason about.

---

## Why Formora?

Most forms fail not because of missing features, but because of **edge cases**:

- async validation results arriving out of order
- array fields shifting indices and breaking errors
- unclear UX rules for when errors should appear
- deeply nested values becoming hard to manage

Formora is built to handle these cases **by design**.
